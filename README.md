# Soundify

Spotify Clone MERN Stack

## Technology Stack

### Frontend

- React
- Vite
- JavaScript
- Tailwind CSS

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose

## Main Features

- User authentication
- Music streaming
- Playlist management
- Search songs, artists, and albums
- Music player
- Admin dashboard
- Song and album management
